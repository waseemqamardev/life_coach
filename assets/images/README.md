# Asset placeholders

Drop your PNG/SVG illustrations from the design files into this folder using the filenames in `lib/core/constants/app_constants.dart`:

### Splash screen (light + dark)
- `splash_image.png` — dark hero (brain + orbit + satellites) **provided**
- `splash_hero_dark.png` — dark hero alternate export
- `splash_hero_light.png` — **light hero** (export with transparent/white background from Figma)
- `wave_light.png` — light bottom wave footer
- `wave_dark.png` — dark bottom glow arc
- `ai_bot.png` — used on processing screen
- `envelope_lock.png` — used on forgot-password screen
- `trophy.png` — used on final-recommendation screen
- `target.png` — used on goal-selection screen
- `safe.png` — used on risk screens
- `empty_state.png` — used on empty-state screen
- `wave_light.png` / `wave_dark.png` — top banner waves
- `user_avatar.png` — fallback profile avatar

If a file is missing, the UI falls back to coloured icon placeholders, so the app still runs.
